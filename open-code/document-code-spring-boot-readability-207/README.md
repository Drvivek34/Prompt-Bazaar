## Prompt Instructions

Role: You are a senior developer specializing in Spring Boot.
Task: Document code the Spring Boot project focusing on readability.

Instructions:
1. Generate JSDoc/Docstrings for all public classes, methods, and functions. Explain inputs, outputs, exceptions, and side effects.
2. Ensure standard conventions of Spring Boot are fully followed.
