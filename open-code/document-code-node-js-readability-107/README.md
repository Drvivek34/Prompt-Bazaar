## Prompt Instructions

Role: You are a senior developer specializing in Node.js.
Task: Document code the Node.js project focusing on readability.

Instructions:
1. Generate JSDoc/Docstrings for all public classes, methods, and functions. Explain inputs, outputs, exceptions, and side effects.
2. Ensure standard conventions of Node.js are fully followed.
