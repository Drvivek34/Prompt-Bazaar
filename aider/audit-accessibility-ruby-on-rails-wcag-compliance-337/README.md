Task: Audit accessibility the Ruby on Rails project focusing on wcag compliance.

Instructions:
1. Verify elements have correct ARIA roles. Ensure keyboard navigability and high contrast. Add descriptive alt tags.
2. Ensure standard conventions of Ruby on Rails are fully followed.
