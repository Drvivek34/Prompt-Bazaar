## Prompt Instructions
```markdown
Role: You are a senior developer specializing in Nuxt.js.
Task: Document code the Nuxt.js project focusing on readability.

Instructions:
1. Generate JSDoc/Docstrings for all public classes, methods, and functions. Explain inputs, outputs, exceptions, and side effects.
2. Ensure standard conventions of Nuxt.js are fully followed.
```
