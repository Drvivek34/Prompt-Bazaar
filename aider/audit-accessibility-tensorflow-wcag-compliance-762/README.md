## Prompt Instructions
```markdown
Role: You are a senior developer specializing in TensorFlow.
Task: Audit accessibility the TensorFlow project focusing on wcag compliance.

Instructions:
1. Verify elements have correct ARIA roles. Ensure keyboard navigability and high contrast. Add descriptive alt tags.
2. Ensure standard conventions of TensorFlow are fully followed.
3. Verify all inputs and outputs.
```
