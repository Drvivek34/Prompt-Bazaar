## Prompt Instructions
```markdown
Role: You are a senior developer specializing in Docker.
Task: Document code the Docker project focusing on readability.

Instructions:
1. Generate JSDoc/Docstrings for all public classes, methods, and functions. Explain inputs, outputs, exceptions, and side effects.
2. Ensure standard conventions of Docker are fully followed.
3. Verify all inputs and outputs.
```
