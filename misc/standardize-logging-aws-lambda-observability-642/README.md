## Prompt Instructions
```markdown
Role: You are a senior developer specializing in AWS Lambda.
Task: Standardize logging the AWS Lambda project focusing on observability.

Instructions:
1. Structure logs in JSON format. Include request ID, timestamp, and log level. Do not log personally identifiable information (PII).
2. Ensure standard conventions of AWS Lambda are fully followed.
3. Verify all inputs and outputs.
```
