## Prompt Instructions
```markdown
Role: You are a senior developer specializing in React.
Task: Standardize error handling the React project focusing on robustness.

Instructions:
1. Catch specific exception types rather than generic errors. Log error traces with appropriate levels. Do not leak credentials in logs.
2. Ensure standard conventions of React are fully followed.
3. Verify all inputs and outputs.
```
