# Contributing to Prompt Bazaar

Add a prompt via the [submission form](https://github.com/Drvivek34/Prompt-Bazaar/issues/new/choose) or a Pull Request.

## Layout
```
<category>/
└── <prompt-name>/
    └── README.md   # name, description, author, target tool, prompt text, source
```

## Each entry must record
- Prompt name + target tool
- Original Author / credit
- Prompt instructions/text
- Source URL of the prompt
- License (if specified)

Part of the [Mega AI Bazaar](https://drvivek34.github.io/Mega-AI-Bazaar/).
